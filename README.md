# gem-uninstall-all

Easy uninstaller for all ruby gems


## Installation

    $ git clone https://github.com/sanadan/gem-uninstall-all.git
    $ cd gem-uninstall-all
    $ sudo make install


## Usage

    $ gem_uninstall-all

If you use rbenv-default-gems and rbenv-gem-rehash, called them.
